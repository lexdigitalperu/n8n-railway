# n8n en Railway con nodos personalizados

Este repo despliega n8n en Railway con nodos permanentes:

- n8n-nodes-serpapi
- n8n-nodes-upload-post
- n8n-nodes-langchain

## Variables Railway

- N8N_HOST=TU_DOMINIO.app
- N8N_PORT=5678
- N8N_PROTOCOL=https
- N8N_WEBHOOK_URL=https://TU_DOMINIO.app/
- WEBHOOK_TUNNEL_URL=https://TU_DOMINIO.app/
